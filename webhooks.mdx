## Webhooks Overview

Webhooks allow you to receive real-time notifications when documentation changes occur in your mdarkplus spaces.

<Callout kind="tip">
  Webhooks are perfect for integrating documentation updates into your CI/CD pipeline or notification systems.
</Callout>

## Setting Up Webhooks

Create and configure webhooks to listen for documentation events.

<Steps>
  <Step title="Create Webhook" icon="plus-circle">
    Go to your space settings and add a new webhook.
  </Step>
  <Step title="Configure Events" icon="settings">
    Select which events trigger the webhook.
    
    <CodeGroup tabs="JSON,Example">
      ```json
{
  "url": "https://your-app.com/webhook",
  "events": ["document.created", "document.updated"],
  "secret": "your-webhook-secret"
}
```
      ```json
{
  "event": "document.updated",
  "document": {
    "id": "doc-123",
    "title": "Updated Guide"
  }
}
```
    </CodeGroup>
  </Step>
  <Step title="Test Webhook" icon="check-circle">
    Send a test event to verify your endpoint.
  </Step>
</Steps>
